الخطوة 1: ضبط Environment Variables
────────────────────────────────────
نسخ الملف:
  copy .env.production.example .env.production

تعديل الإعدادات:
  افتح .env.production
  غيّر VITE_SITE_URL إلى رابط موقعك الفعلي


الخطوة 2: Build المشروع
────────────────────────
  npm run build:prod


الخطوة 3: الرفع
────────────────

الخيار A: Vercel (موصى به)
  npm install -g vercel
  vercel --prod

الخيار B: سيرفر خاص
  - ارفع محتويات dist على public_html
  - تأكد من وجود .htaccess


الخطوة 4: الاختبار
────────────────────
1. افتح الموقع وتأكد أنه يعمل
2. اختبر المشاركة:
   https://developers.facebook.com/tools/debug/
